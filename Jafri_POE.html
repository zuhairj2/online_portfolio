Aerodynamic Design Project

January 2019, Principles of Engineering

Click here to see our Project Notebook!

Here is the Project Rubric

The objective of this project was to build and test different projectile designs, to find the farthest traveling rocket that can be used by Nitro Planes. I created the whole rocket, which is able to go 88 yards in 3.99 seconds. My first prototype was a failure because of the fact that I wasn't familiar with what to do. Once we were able to redo the rocket, I knew all the components you need to make a rocket. I was able to complete the rocket with all the requirements and it passed the minimum requirement with flying colors. I never gave up on my rocket, which enabled me to make all the adjustments I needed. I learned that you need a light material to create a rocket so that it can go farther and faster without the rocket weight itself down. I used 3 small fins on the back as a stabilizer, which helped immensely. Overall, I enjoyed this project, it's one of the best projects of the year so far.
